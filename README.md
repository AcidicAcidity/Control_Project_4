# Это репозиторий для контрольной №4 по дициплине "Технологии Индустриального Программирования"
# Счетчик нажатий

Это простой веб-проект, представляющий собой страницу с кнопками для увеличения и уменьшения значения счетчика. Проект предназначен для демонстрации базовой работы с DOM и событиями в JavaScript.

## Описание работы

На странице отображается число — текущее значение счетчика, изначально равное нулю. Кнопки "+" и "-" увеличивают или уменьшают это значение соответственно. При каждом нажатии число на странице обновляется без перезагрузки.

## Технологии

- HTML для разметки страницы
- CSS для оформления в приятных зеленых тонах с градиентами и эффектами на кнопках
- JavaScript для обработки нажатий и динамического изменения текста счетчика

## Как использовать

- Откройте файл в любом современном браузере.
- Используйте кнопки "+" и "-" для изменения значения счетчика.

## Цель проекта

Демонстрировать простую интерактивность веб-страницы с использованием клиентского JavaScript и тренировать навыки вёрстки и скриптинга.
